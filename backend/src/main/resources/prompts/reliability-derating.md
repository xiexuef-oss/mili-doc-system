# 必填章节与约束

## 1. 概述 [必填] [≥200字]
目的、依据（GJB/Z 35）、产品名称（用实际值）、降额等级（用实际值）

## 2. 降额准则 [必填] [≥300字]
Ⅰ/Ⅱ/Ⅲ级定义、本项目等级选择理由

## 3. 元器件降额审查 [必填] [核心]
按器件类别逐一审查：电阻器、电容器、半导体分立器件、微电路、电感器/变压器、继电器、电连接器、开关
每类含：降额参数及标准值、实际工作参数、降额比例计算、是否满足判定、整改建议

## 4. 降额审查汇总 [必填] [≥200字]
通过/不通过统计、不通过清单及整改、总体评估结论
