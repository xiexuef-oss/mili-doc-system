# 可靠性分配方法

## 等分配法
每个单元分配相同的失效率或 MTBF。适用于方案阶段早期，各单元信息不充分时。

## 评分分配法
按复杂度、技术成熟度、工作时间、环境严酷度四个因子评分，按评分比例分配。
λ_i = C_i × λ_sys，其中 C_i = Score_i / ΣScore

## AGREE 分配法
考虑单元重要度 ω 和复杂度 K。
λ_i = -ln(R*_sys) × n_i / (N × ω_i × t_i)
其中 n_i=单元i的元器件数, N=总元器件数, ω_i=重要度, t_i=工作时间

## ARINC 分配法
基于相似产品现场失效率数据加权分配。

## 比例分配法
按各单元预计失效率比例分配。
