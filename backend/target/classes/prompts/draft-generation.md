# 生成约束

1. 章节标题：## 用于一级章，### 用于二级节，标题后换行正文另起
2. 锁定已知值：项目上下文中"已知数据"的数值和事实原样使用
3. 标记不确定：用 `[⚠待核实: 具体说明]` 标记
4. 禁止占位符：禁止"XXX"、"待补充"、"略"、"TBD"、"详见XX"
5. 每章输出 AI_META：
<!-- AI_META
章节: [标题]
完成度: [0-100]
待核实: [- 事项]
确定项: [- 事项]
缺项: [- 事项]
-->
6. 不输出开场白和结束语
