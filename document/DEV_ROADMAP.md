# 军工项目文档策划与编制一体机 - 开发任务清单

## 已完成 ✓

### 后端 (22 控制器)
- 项目框架 (Spring Boot 3.2.5 + MyBatis-Plus + PostgreSQL)
- 认证授权 (JWT + Spring Security + BCrypt)
- 系统管理: 用户/角色/权限/字典 CRUD + RBAC 绑定
- 项目管理: Project/ProjectMember/ProjectStage/StageTransitionCheck (全部含 Controller)
- 文档模块: DocFile/DocCatalog/DocVersion/DocEditSession/DocEditLock/DocChangeImpact/DocEffectiveBaseline (全部含 Controller)
- 评审模块: ReviewMeeting/ReviewMeetingDocument/ReviewExpertOpinionFile/ReviewMeetingOpinionFile (全部含 Controller)
- 模版库: DocTemplate CRUD + 文件上传/下载 + 变量管理
- 标准库: Standard CRUD + StandardClause CRUD + 关键字搜索 + 文件管理
- 文件存储: LocalFileStorage (MinIO 可插拔) + FileController 下载端点

### 前端 (28 页面)

| 模块 | 页面 |
|------|------|
| **基础** | 登录 / 主布局(侧边栏导航) |
| **项目管理** | 项目列表 / 项目详情 / 项目成员 / 项目阶段 / 阶段转阶段检查 |
| **文档管理** | 文档管理 / 文档目录 / 文档版本 / 编辑会话 / 编辑锁定 / 生效基线 / 变更影响分析 |
| **评审管理** | 评审会议 / 评审文档关联 / 专家意见 / 会议意见汇总 |
| **模版库** | 模版管理(含上传+变量配置) / 从模版创建文档 |
| **标准库** | 标准库(含条款管理) / 标准详情 |
| **系统管理** | 用户 / 角色 / 字典 / 权限 |

### 基础设施
- 24 张表 + 种子数据 / RBAC 完整绑定 / ngrok 外网访问 / Gzip 预压缩
- 文件存储: 本地 + MinIO 条件注入
